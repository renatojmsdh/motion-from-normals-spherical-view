ppv

SYNOPSIS

function A = ppv(v) ;

DESCRIPTION 

 PPV  Antisymmetric matrix for cross product.

   A = ppv(v) returns the antisymmetric matrix A 
   such that A*w = v x w is the cross product of 
   the vectors v and w. v must be 3 element vector.

   See also vpp.
